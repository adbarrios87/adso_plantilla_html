Aqui se daran las indicaciones para el manejo de git

CLONAR Y CONFIGURAR GIT PROYECTO:
Paso 1: Crear en GitHub el repositorio
Paso 2: Ir a Git Bash y ubicar el workspace en la carpeta de trabajo (cd - para entrar a carpeta, ../ retroceder en carpetas, clear - limpiar consola)
Paso 3: git clone [url-repositorio] : para descargar el proyecto de manera local
Paso 4: Creo un archivo txt "readme.txt"
Paso 5: Abrir el archivo TXT con Visual Studio Code
Paso 6: Copiar carpeta del proyecto al explorador de VSC
Paso 7: Abrir nuevo terminal 
Paso 7: git status : para ver el estado del proyecto
Paso 8: git add. : para agregar el archivo al proyecto
commit: es un paquete de cambios
Paso 9: git commit -m "cambio": para indicar el cambio realizado
Paso 10: git push: para subir el cambio al remote

Para crear ramas (para ver estados anteriores de cambios) git checkout  -b feature_fecha

CREAR NUEVO PROYECTO
Paso 1: Crear un nuevo documento "index.html"
Paso 2: Crear carpeta src y subcarpetas: assets, css, js
